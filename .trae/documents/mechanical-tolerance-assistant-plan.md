# 机械公差助手 - 实施计划

## 摘要

创建一个单文件 Streamlit Web 应用 `app.py`（约 2500-3500 行），实现机械公差相关的 4 大核心功能：尺寸链计算（极值法）、公差查询、公差配合推荐、形位公差推荐。严格遵循 GB/T 1800.1-2020、GB/T 1801-2009、GB/T 1184-1996 国家标准，所有数据硬编码，无需外部数据文件。

## 当前状态分析

- 无现有代码，从零开始
- 已完成标准数据研究（IT01-IT18 公差值表、基本偏差公式、常用配合代号、形位公差值表、推荐规则）
- 技术栈：Python 3.10+ / Streamlit / Pandas / NumPy

## 实施方案

### 具体实施步骤

#### 步骤1: 创建文件骨架 + 导入 + 页面配置 + CSS
- 文件路径: `/workspace/app.py`
- 设置 `st.set_page_config(page_title="机械公差助手", layout="wide")`
- 定义自定义 CSS（表格美化、卡片样式、标题居中）
- 导入 streamlit, pandas, numpy, math, re

#### 步骤2: 硬编码标准公差数据表
- **IT01-IT18 公差值表**: 13个尺寸段 × 20个等级，单位微米
- **形位公差值表**（GB/T 1184-1996）:
  - 圆度/圆柱度: 13个直径段 × 13个等级(0-12)
  - 直线度/平面度: 10个长度段 × 12个等级(1-12)
  - 平行度/垂直度/倾斜度: 10个长度段 × 12个等级
  - 同轴度/对称度/跳动: 10个直径段 × 12个等级
- **常用配合数据库**: 20+种配合的代号、类型、描述、应用场景
- **配合推荐规则**: 15+条规则映射场景参数到推荐配合
- **形位公差推荐数据库**: 按零件类型(轴/孔/箱体/法兰/端盖)和功能分类

#### 步骤3: 实现基本偏差计算函数
- `get_size_range_index(nominal_size)` → 返回尺寸段索引
- `get_geometric_mean(nominal_size)` → D = √(D1×D2)
- `get_it_value(nominal_size, grade_str)` → 查表返回IT值(μm)
- `calc_shaft_fundamental_deviation(letter, D, it_grade)` → 实现 a~zc 的公式
  - 关键公式: d=-16D^0.44, e=-11D^0.41, f=-5.5D^0.41, g=-2.5D^0.34, h=0, n=+5D^0.34 等
  - 边界条件: a在D=120切换, b在D=160切换, c在D=40切换, s在D=50切换
- `calc_hole_fundamental_deviation(letter, D, it_grade, nominal_size)` → 通用规则+特殊规则
  - 通用: EI=-es(A~H), ES=-ei(K~ZC)
  - 特殊: J/K/M/N(IT≤8), P~ZC(IT≤7), 尺寸>3mm时 ES=-ei+Δ, Δ=ITn-IT(n-1)
- `query_tolerance(designation, nominal_size)` → 返回完整公差信息字典

#### 步骤4: 实现尺寸链计算模块
- 极值法公式:
  - A₀ = Σ(ξᵢ × Aᵢ)，ξ=+1(增环), ξ=-1(减环)
  - ES₀ = Σ(ξᵢ × ESᵢ), EI₀ = Σ(ξᵢ × EIᵢ)
  - T₀ = Σ|ξᵢ| × Tᵢ
- 输入验证: 尺寸>0, 上偏差≥下偏差
- 输出: 封闭环基本尺寸/上下偏差/公差, 各环传递系数和贡献率
- 显示完整计算公式和过程

#### 步骤5: 实现公差查询模块
- 解析公差带代号(如H7, f6, JS7)
- 计算上下偏差、公差值、极限尺寸
- 配合分析: 最大/最小间隙或过盈, 配合类型判定
- 显示3-5个典型应用场景
- 公差带示意图(可选, 用matplotlib)

#### 步骤6: 实现公差配合推荐模块
- 规则引擎: 根据载荷/转速/精度/拆卸需求匹配规则
- 输出2-3个推荐配合, 每个含优缺点和适用情况
- 推荐表面粗糙度和形位公差
- 支持基孔制/基轴制选择

#### 步骤7: 实现形位公差推荐模块
- 按零件类型+功能+精度等级推荐
- 查询具体公差值
- 说明标注位置和方法

#### 步骤8: 实现主界面
- 4个tabs布局
- 顶部系统名称+版本号
- 底部免责声明
- 复制结果按钮
- 输入验证和错误提示

### 关键决策与假设

1. **数据来源**: IT公差值和基本偏差公式基于 ISO 286 / GB/T 1800.1-2020，通过公式计算+查表双重验证
2. **公差带图**: 暂不绘制matplotlib图表，用文字和表格清晰展示结果（减少依赖复杂度）
3. **配合推荐**: 采用规则匹配+评分排序，非机器学习
4. **形位公差**: 仅覆盖0-500mm常用尺寸范围
5. **js偏差**: 对称分布, ES=+IT/2, EI=-IT/2
6. **j偏差**: 无简单公式，使用查表法（需额外硬编码j5-j8的偏差值）
